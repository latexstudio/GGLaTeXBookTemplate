# GGLaTeXBookTemplate
LaTeX template for books, manuals, documentations, etc.

## Compiling
This LaTeX document is tested on [ShareLaTeX](https://www.sharelatex.com/). 

The used compiler is **XeLaTeX**.

The main document is **main.tex**.
